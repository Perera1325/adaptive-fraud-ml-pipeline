Adaptive Fraud ML Pipeline



Overview

This project builds a self-evolving machine learning pipeline for detecting phishing and financial fraud using multimodal data.



Key Features

\- Multimodal fraud detection (URLs, emails, transactions)

\- Adaptive retraining pipeline

\- Explainable AI (SHAP)

\- Real-world simulation of data drift



Tech Stack

\- Python

\- Scikit-learn

\- TensorFlow / PyTorch

\- SHAP

\- Pandas / NumPy



Pipeline Stages

1\. Data Collection

2\. Data Preprocessing

3\. Feature Engineering

4\. Model Training

5\. Evaluation

6\. Deployment \& Monitoring
